The Problem

Clients struggled to find reliable craftsmen in Syria, while skilled workers lacked a platform to showcase their services. Existing solutions were either too generic or lacked essential features like location-based search and secure booking.

The Solution

JobScope is a comprehensive web system that connects service seekers with craftsmen in Syria. Built advanced search with map-based filters, profession selection, and range detection. Designed a booking and order management system with real-time status updates. Added multi-criteria rating and review features.

Challenges & Solutions

  • Implementing accurate location-based search with Leaflet.js
  • Creating secure authentication system with JWT
  • Designing effective booking and order management workflow
  • Building multi-criteria rating system with automatic calculations

Lessons Learned

  • Complexity of marketplace platform development
  • Importance of user verification and trust systems
  • Effective use of mapping libraries for location services
  • State management with Zustand for complex applications
